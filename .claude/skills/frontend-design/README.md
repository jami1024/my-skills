# Frontend Design Skill

创建独特、生产级的前端界面，避免通用的 AI 美学风格。

## 快速开始

### 触发方式

在对话中提到以下关键词会自动激活此 skill：
- "UI 设计"
- "前端设计"
- "界面美化"
- "frontend-design"
- "landing page"
- "落地页设计"

或直接使用命令：
```
/frontend-design
```

## 核心理念

### 避免 AI 通用美学

❌ **不要**
- Inter / Roboto / Arial 字体
- 紫色渐变 + 白色背景
- 预测性的组件布局
- 缺乏特色的"安全"设计

✅ **要**
- 独特且有个性的字体选择
- 大胆的主导色 + 强烈点缀
- 不对称、重叠、对角线布局
- 针对上下文的独特设计

### 美学方向

选择一个明确的美学方向并精确执行：

| 风格 | 适用场景 |
|------|----------|
| 极简主义 | 高端品牌、艺术画廊 |
| 最大化主义 | 创意机构、娱乐平台 |
| 复古未来 | 游戏、音乐、科技 |
| 奢华精致 | 奢侈品、高端服务 |
| 野蛮主义 | 实验性项目、艺术 |
| 编辑杂志 | 媒体、出版、博客 |
