# 学习文档模板

直接复制使用，替换 `{占位符}` 即可。

---

## 学习规划模板

```markdown
# {技术主题} 学习规划

## 学习目标
{一句话说明学完后能做什么。}

## 前置知识
- {需要提前掌握的内容}

## 知识点清单

### 阶段一：{阶段名}（基础）
1. {知识点} — {一句话说明}
2. {知识点} — {一句话说明}

### 阶段二：{阶段名}（核心）
3. {知识点} — {一句话说明}
4. {知识点} — {一句话说明}
5. {知识点} — {一句话说明}

### 阶段三：{阶段名}（实践与进阶）
6. {知识点} — {一句话说明}
7. {综合项目} — 串联以上所有知识点

## 配套代码示例
- `examples/01-{名称}/main.{ext}` — {说明}
- `examples/02-{名称}/main.{ext}` — {说明}
- `examples/03-{名称}/main.{ext}` — {说明}

## 预计学习时间
- 阶段一：{X} 小时
- 阶段二：{X} 小时
- 阶段三：{X} 小时
```
