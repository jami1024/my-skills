## 示例填写

```markdown
**项目名称**: FinFlow Dashboard
**项目类型**: Web 应用 - 金融数据仪表板

### 品牌定位
- **行业**: 金融科技
- **品牌个性**: 专业可靠、创新前沿
- **竞争对手**: Stripe Dashboard, Linear, Notion

### 目标受众
- **主要用户**: 中小企业财务人员
- **用户目标**: 快速查看财务概况、生成报表
- **技术熟练度**: 普通

### 美学方向
- 期望感觉: 专业的、值得信赖的、简洁的
- 美学风格: 极简主义
- 颜色偏好: 深色主题、高对比度

### 技术栈
- React + TypeScript
- Tailwind CSS + shadcn/ui
- 需要响应式设计
```
